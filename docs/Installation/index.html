1. Installation

lakefile.lean:

require Png from git "https://github.com/alok/lean4-infoview-png" @ "main"

lakefile.toml:

[[require]]
name = "Png"
git = "https://github.com/alok/lean4-infoview-png"
rev = "main"

Then run lake update.
